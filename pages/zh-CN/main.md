# Emakefun DM11 Arduino库

## 介绍

DM11是Emakefun开发的使用I2C协议驱动4路PWM从而驱动2路电机的模块，默认I2C地址为0x15。

| DM11库支持的开发板系列 |
| --- |
| Arduino UNO |
| Arduino Nano |
| Arduino Mega 2560 |
| ESP32 |
| ESP32S3 |
